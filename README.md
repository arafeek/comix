# COMIX

### The hell?
  Comix is a light-hearted command line tool for quickly viewing sale items on 
  [comixology](https://www.comixology.com). Apologies in advance if it violates some kind
  of EULA :(

### Usage 
 - Step 1:     
` > npm install -g comix `

 - Step 2:     
` > comix  `

 - Step 3:
